1. VOLLEYBALL MANAGER STARTEN
Der Volleyball Manager besteht aus dem Server und dem Administrator zur Verwaltung.
Der Server speichert alle Informationen und muss deswegen unbedingt vor dem
Administrator gestartet werden.

1.1 SERVER STARTEN
Klicken Sie dazu auf das Symbol des Servers auf Ihrem Desktop oder im Startmenü.
Darauf erscheint eine schwarze Box, in der der Server geladen wird. Warten Sie, bis
keine weiteren Veränderungen in der Box erscheinen und die Zeile Started in
[..:..]ms erscheint. Die Dauer des Starts kann, abhängig von der Geschwindigkeit
Ihres Rechners, zwischen 10 sec und 2 min liegen.

1.2 ADMINISTRATOR STARTEN
Nachdem Sie den Server gestarted haben, können Sie den Administrator öffnen. 
Klicken Sie dazu wiederum auf das entsprechende Symbol auf dem Desktop oder im
Startmenü.

2. VOLLEYBALL MANAGER TESTEN
Nachdem Sie den Administrator gestartet haben, wählen Sie im Menü "Hilfe->Hilfe".
Lesen Sie bitte in der Hilfe die Seite "Einleitung->Schnellstart".
